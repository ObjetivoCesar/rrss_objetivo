# Guía de Implementación: Alimentando el Prompt con tu ADN

Para que el **Prompt Maestro** funcione con la máxima efectividad y logre ese efecto de "me habla a mí", es crucial que la información que extraigas de tu documento de ADN sea precisa y esté bien estructurada. Sigue estos pasos para preparar los datos antes de usarlos en el prompt.

## Paso 1: Extracción de Datos del ADN

Revisa tu documento de ADN y busca las secciones que correspondan a las variables del prompt. No copies todo el documento; selecciona solo lo más relevante para el tema específico del video que quieres crear.

| Variable del Prompt | Qué buscar en tu ADN | Ejemplo de Extracción |
| :--- | :--- | :--- |
| **Descripción del Avatar** | Busca el perfil demográfico y psicográfico. | "Emprendedores digitales de 25-40 años que venden servicios de consultoría." |
| **Principales Dolores** | Busca las frustraciones que mencionan tus clientes. | "Sienten que sus contenidos no generan ventas, solo likes." |
| **Principales Deseos** | Busca los objetivos que quieren alcanzar. | "Quieren un sistema predecible de atracción de clientes." |
| **Objeciones Comunes** | Busca los "peros" que te han puesto. | "No tengo tiempo para crear contenido todos los días." |
| **Lenguaje Clave** | Busca palabras o frases exactas que ellos usen. | "Contenido de relleno", "vender sin vender", "quemar la audiencia". |

## Paso 2: Preparación del Contexto

Antes de pegar el prompt en tu herramienta de IA (como ChatGPT, Claude o Manus), asegúrate de tener claros el **Tema del Video** y el **Concepto Clave**.

*   **Tema del Video:** El área general (ej. "Estrategia de Instagram").
*   **Concepto Clave:** Lo que vas a enseñar específicamente (ej. "Cómo usar las historias para cerrar ventas").

## Paso 3: Ejecución del Prompt

Copia el **Prompt Maestro** y rellena las variables con la información que extrajiste en el Paso 1. Al final del prompt, añade el tema y el concepto clave del video actual.

> **Consejo Pro:** Si tu documento de ADN es muy extenso, puedes pedirle a la IA primero: *"Analiza este documento de ADN y extrae los dolores, deseos y lenguaje clave de mi avatar para un video sobre [TEMA]"*. Luego, usa esa salida para rellenar el Prompt Maestro.

## Paso 4: Revisión del Guion Generado

Una vez que la IA genere el guion, léelo en voz alta. Si sientes que "te habla a ti" (como el video original), entonces el ADN está bien integrado. Si suena genérico, revisa si el **Lenguaje Clave** y los **Dolores** que proporcionaste son lo suficientemente específicos.
